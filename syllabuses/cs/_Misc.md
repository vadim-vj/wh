## Функции

### Линейные функции и уравнения
- `y = ax + b`, коэффициент при переменной ненулевой (иначе функция-константа)
- функции с `b = 0` называются *однородными*
- форма записи уравнения `ax + by + c = 0` называется *общей*
- коэффициент при переменной (`a`) равен тангенсу (в радианах? да) угла наклона графика функции
- при `a > 0` угол с осью `x` острый, при `a < 0` - тупой
- коэфф. `b` равен значению функции при `x = 0` - точке пересечения графика с осью ординат. При `b = 0` прямая проходит через начало координат
- точка пересечения с осью `x` (нуль функции) также одна, и равна `-b/a`
- угол между двумя прямыми: `tg(alph) = abs((a1 - a2) / (1 + a1 * a2))`
- обратная функция - тоже линейная: `x = 1/a * y - b/a`

### Квадратичные функции и уравнения
- коэфф.при квадрате (`a`) ненулевой
- любой коэфф. может быть дробным
- ветви параболы направлены вверх если `a > 0` и вниз иначе
- чем больше `a`, тем круче парабола, а чем меньше - тем она более широкая и плоская (хотя при `a > 1` она уже всегда крутая)
- положительные коэффициенты `b` сдвигают параболу влево и вниз, отрицательные - вправо и вниз (при отрицательном `a` вправо и вверх и влево и вверх)
- `c` просто двигает по оси ординат вверх-вниз, независимо от знака `a`
- точка пересечения с осью `y` находится элементарно (подставить `x = 0`), а 1-2 точки пересечения с осью `x` - через приравнивание функции к нулю и решение квадратного уравнения
- выражение `D = b^2 - 4ac` называется *дискриминантом* - произведение (в общем случае вычисляемое по сложной формуле) коэффициентов многочлена. Определяет количество корней многочлена: равен нулю тогда и только тогда, когда есть кратные корни
- корни квадратного уравнения: `(-b + sqrt(D))/2a` и `(-b - sqrt(D))/2a` (совпадают при нулевом дискриминанте) - точки пересечения с осью `x`
- вершина параболы - среднее арифметическое корней (из-за симметричности функции) `x = -b/2a`
- значение `y` в вершине можно получить подстановкой или запомнить (`y = -D/4a`)
- функция не строго монотонна, поэтому обратной ф. нет. Но можно рассматривать пару корней. Выразить их можно отсюда: `y = a(x - b)^2 + c`

### Кубические функции и уравнения
- график - кубическая парабола, ветви разнонаправлены. Симметрична относительно точки перегиба (`x = -b/3a`)
- коэффициент при кубе определяет направления левой и правой ветви: `a > 0` - правая вверх, левая вниз, `a < 0` - наоборот
- коэфф. при квадрате и линейном члене дают доп. выпуклости вблизи нуля
- имеет 2 точки локального экстремума (минимума и максимума)
- кубическое уравнение имеет 3 корня, существует несколько способов решения (типа формулы Кардано)

### Показательные функции и степени
- *возведением в степень* `n` называется умножение числа `a` на себя `n` раз (для натуральных степеней)
- для рациональных степеней операция выражается через корень n-ной степени: `a^(p/q) = root q(a) ^ p`. 
- *показательная функция* - функция вида `y = a^x`, где `a` называется *основанием степени*, а `x` - *показателем степени*
- показательная функция с основанием `e` называется *экспонентой*
- свойства степеней: `a^0 = 1`, сумма степеней: `a^(x+y) = a^x * a^y`, разность степеней: `a^(x-y) = a^x / a^y`, `(a^x)^y = a^(x*y)`, степень произведения: `(ab)^x = a^x * b^x`, степень частного: `(a/b)^x = a^x / b^x`
- первая степень любого числа равна самому числу, нулевая - единице
- отрицательная степень: `a^-x = 1/a^x`
- обратная функция - логарифм: `y(x) = a^x`, `x(y) = log a(y)`, `y(x(y)) = a^(log a (y)) = y`
- в *степенной* функции основание степени является переменной (напр., `x^2`), тогда как в показательной - показателем (напр., `2^x`)
- показательная функция с основанием меньше единицы убывает, с б*о*льшим - возрастает. Возрастание (убывание) асимптотическое, в бесконечность (ноль) при стремлении `x` в бесконечность
- график пересекает ось `y` в точке `[0, 1]`, в точке `x = 1` функция равна основанию
- для положительных оснований функция непрерывна, монотонна и строго больше нуля. Для отрицательных ничего из этого не выполняется

### Логарифмы и логарифмические функции
- *логарифмом* `b` по основанию `a` называют число, в которое нужно возвести `a`, чтобы получить `b`: `log a(b)`. По определению, это функция, обратная показательной
- логарифм не определен при нулевом или отрицательном основании `a`
- *основное логарифмическое тождество*: `a ^ log a(b) = b`. Следствие - из равенства логарифмов следует равенство логарифмируемых выражений
- смена основания показательной функции: `a^x = b^(x * log b(a))` (потому что `b^log b(a) = a`)
- свойства: произведение - `log(xy) = log(x) + log(y)`, частное - `log(x/y) = log(x) - log(y)`, степень: `log(x^p) = p * log(x)`
- из свойств логарифмов вытекает возможность упрощения мат. операций для больших чисел: вместо сложения/деление/возведения в степень можно складывать/вычитать/умножать логарифмы чисел (например, по таблицам; по ним же можно и восстанавливать значение логарифмируемого выражения) 
- область определения логарифмической функции строго больше нуля. При основании больше единицы функция возрастает, меньше - убывает
- график логарифма уходит в +/- бесконечность в точке `x = 0`, а в точке `x = 1` пересекает ось `x` (`y(1) = 0`). В точке, равной основанию (`x = a`) функция равна единице: `y(a) = 1`
- натуральный логарифм распространен во многом потому, что его производная особенно проста: `ln'(x) = 1/x` (площадь под гиперболой)

## Геометрия

### Углы и окружности
- площадь круга - `pi * R^2`
- длина окружности - `2 * pi * R`
- *градус* - единица измерения плоских углов, 1/360 часть окружности
- *радиан* - угол, соответствующий дуге, длина которой равна её радиусу
- т.к. длина окружности равна `2 * pi` радиусов, то в окружности `2 * pi` радиан, т.е. угол в 180 градусов = `pi` радиан
- формула перевода градусов в радианы: `a(deg) = a(rad) * (180 / pi)`
- формула перевода радиан в градусы: `a(rad) = a(deg) * (pi / 180)`

### Тригонометрические функции
- *синус* - отношение противолежащего, *косинус* - прилежащего катета к гипотенузе
- удобно рассматривать на окружности единичного радиуса, тогда гипотенуза всегда = `1` и ее можно не учитывать
- график синуса проходит через точку `[0, 0]` (противолежащий углу `0` катет также равен нулю), косинуса - через `[0, 1]` (прилежащий катет нулевого угла равен гипотенузе)
- оба графика периодичны, ограничены значениями `y [1, -1]`. `y` обращается в ноль через каждые `pi` радиан (каждые `180` градусов)
- для углов в `30`/`60` градусов значения на осях равны `1/2` (это и есть синус/косинус), из теоремы Пифагора можно вычислить значения парной функции. По ней же - для `45`
- отношения катетов - это *тангенс* (`tg`, противолежащего к прилежащему) и *котангенс* (`ctg`, наоборот). Из-за сокращения гипотенузы в пропорциях они равны отношениям `sin/cos` и `cos/sin` соответственно
- графики тангенса/котангенса периодичные, с разрывами в точках, где `cos`/`sin` (соответственно) обращаются в нуль. в этих точках функции асимптотически уходят в бесконечность. У тангенса правая ветвь обращена вверх, левая вниз, у котангенса наоборот
- обратные тригонометрические функции имеют префикс *арк-*, например, `arcsin(sin(x)) == x`
- множества разных тождеств, вроде `sin^2 + cos^2 = 1` и `tg * ctg = 1`

### Площади фигур
- *изопериметрическая задача* - поиск фигуры, имеющей наибольшую площадь при заданном периметре. *Изопериметрическое неравенство* утверждает, что `4 * pi * S <= L^2`. Для гладких кривых такой фигурой является круг (неравенство обращается в равенство)
- площадь треугольника равна половине произведения его основания на высоту: `S = (a * h) / 2`
- площадь прямоугольного треугольника - половина произведения катетов: `S = (a * b) / 2`
- площадь трапеции - половина произведения суммы оснований на высоту: `S = (a + b) * h / 2`
- площадь параллелограмма равна произведению стороны на высоту, проведенную к этой стороне: `S = a * h`

## Дроби и их преобразования

### НОД и НОК
- *НОД* (greatest common divisor) - самый большой из делителей. Существует и однозначно определен, если хотя бы одно число не равно нулю. Чисел не обязательно два, может быть больше
- *НОК* (least common multiple) - наименьшее натуральное число, которое делится на `m` и `n` без остатка
- для нахождения НОД существуют алгоритм Евклида и бинарный алгоритм `<!-- к бинарному поиску можно позже вернуться -->`
- НОК равен `abs(m * n) / НОД`
- НОК взаимно простых чисел равно модулю их произведения: `[n, m] = n * m`
- произведение НОД и НОК: `(n, m) * [n, m] = n * m`
- *основная теорема арифметики* утверждает что любое натуральное число можно единственным способом разложить на произведение простых чисел. Представление числа в виде произведения простых чисел в степенях называется *каноническим разложением* `<!-- вернуться -->`
- если для чисел известно их каноническое разложение, вычислить НОК и НОД можно просто последовательно перебирая простые числа, и беря минимальную (для НОД) или максимальную (для НОК) степень для каждого сомножителя

### Обыкновенные и десятичные дроби
- *правильная* дробь по модулю меньше единицы
- обыкновенная дробь, содержащая только числитель и знаменатель, называется *простой*
- в записи *смешанной* дроби перед дробью пишется целое число. Это обозначает сложение (напр., `2 + 3/7`). Использовать такую запись не рекомендуется, из-за возможной путаницы с умножением
- *составная* дробь содержит несколько черт (символов деления). Также иногда называется "многоэтажной"
- домножение и числителя и знаменателя на одно и то же число не меняет значение дроби. Верно и обратное - при сокращении на общий делитель значение дроби не меняется
- у *несократимой* дроби числитель и знаменатель *взаимно просты*. Для приведения к такому виду числитель и знаменатель нужно поделить на их НОД
- для сравнения, сложения и вычитания дробей их приводят к общему знаменателю: находят НОК знаменателей `M = [d1, d2]` и домножают числитель и знаменатель первой дроби на `M/d1`, и второй на `M/d2`. После этого можно выполнять действия над числителями, знаменатель остается общим
- при умножении перемножаются числители и знаменатели: `a/b * c/d = (a * c)/(b * d)`. Деление ~ умножению на обратную дробь
- целое число `n` можно представить как `n/1`. То есть при умножении на него домножается только числитель, а при сложении нужно приводит к общему знаменателю
- запись десятичной дроби однозначна (в отличие от обыкновенной), но имеются исключения, например, `0.999999... == 1`
- перевод обыкновенной дроби в десятичную - последовательное деление столбиком `<!-- вернуться -->`

## Числа

### Выполнение операций "в столбик"
- сложение: с младших разрядов, по одной цифре, в случае результата больше 10 - перенос (указываем его над первым слагаемым) и сложение с ним в следующем разряде
- вычитание: также, с младших разрядов, по одной цифре. В случае если уменьшаемое меньше вычитаемого, "занимаем" один десяток из старших разрядов (ставим над ними точку). Из нуля занимать нельзя, поэтому ищем первый старший ненулевой разряд, и занимаем из него, отмечая точкой все до него. Заем одного десятка из нуля превращает его в 9
- умножение: поразрядное, начиная с младшего, умножение каждой цифры второго числа на каждую цифру первого. Десятки переносятся (записываются над следующим разрядом первого сомножителя) и складываются с результатами умножения в следующем разряде. (Общие) результаты умножения для каждой цифры записываются один под другим, со сдвигом не разряд, после чего складываются (в столбик)
- деление: начиная со старших разрядов, ищется минимально количество цифр, результат деления которого ("неполное частное") больше единицы. Неполное частное записывается как цифра в результат, а к остатку от его деления дописываются следующие разряды и повторяют процесс. Если в результате получили число меньше делителя - то это остаток

### Системы счисления
- *система счисления* - это правило записи чисел с помощью заданного набора специальных знаков - цифр
- максимальная цифра в позиционной системе счисления на единицу меньше основания
- перевод из одной системы в другую как правило делается с промежуточным переводом в десятичную, чтобы не выполнять арифметические операции по правилам систем с другим основанием
- перевод в десятичную - умножение цифр числа на основание СС в степени позиции (т.к. операции выполняются по правилам десятичной арифметики)
- перевод из десятичной - последовательным (с остатком, деля частные на каждом шаге) делением числа на основание новой системы счисления. Остатки будут цифрами нового числа, начиная с младшего разряда (т.е. запись в обратном порядке). Очевидно, что остаток от деления (цифра) всегда меньше основания СС)
- перевод из 8/16-ричных в двоичную - замена (по таблице) каждой цифры на двоичное представление. Нули слева в каждой группе должны присутствовать (размер двоичных групп - 3/4 цифры соответственно)
- перевод из двоичной в 8/16-ричную - разбивка числа на группы цифр по 3/4 соответственно, и замена (по таблице) каждой группы на цифру в нужной системе

### Представление в памяти
- *число с плавающей точкой* - экспоненциальная форма представления вещественных (действительных) чисел, в которой число хранится в виде мантиссы и порядка (показателя степени)
- число с плавающей точкой имеет *нормализованную* форму, если его мантисса лежит в интервале `0.1 <= M < 1` (напр., `0.1234*10^4`, `0.3456*10^-1`)
- *FLOPS* - "FLoating-point Operations Per Second", произносится как *флопс* - число операций с плавающей точкой в секунду, которое может выполнить данная система
- *машинный эпсилон* - наименьшее положительное число такое, что `1 + e = 1`; то есть минимальное число, различимое машиной, точность операций с плавающей точкой. Два числа `a` и `b` неразличимы, если `1 < a/b < 1 + e`
