# Markdown
## Headers (up to 6 levels)
```
Setex             Atx                   HTML
-------------------------------------------------------
Level1     or     # Level1       -->    <h1>Level1</h1>
======

Level2     or     ## Level2      -->    <h2>Level2</h2>
------
...

N/A               ###### Level6  -->    <h6>Level6</h6>
```
## Emphasis
```
                                        HTML
-------------------------------------------------------
**Bold**   or     __Bold__       -->    <strong>Bold</strong>

*Italic*   or     _Italic_       -->    <em>Italic</em>
```
