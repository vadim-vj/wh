Оф. сайт компании: https://numenta.com

Моделирование неокортекса. Базовая технология - иерархическая временная память (Hierarchical temporal memory, HTM)

Коммерческие проекты:

  • HTM Studio - поиск аномалий в данных ( https://numenta.com/applications/htm-studio)
  • Rogue Behavior Detection
  • Geospatial Tracking
  • HTM for Stocks
  • Numenta Anomaly Benchmark (NAB) - поиск аномалий в данных ( https://numenta.com/applications/numenta-anomaly-benchmark
  • Grok - поиск аномалий в данных ( http://www.grokstream.com)
  • Cortical.io - обработка естественного языка ( http://www.cortical.io)

OpenSource-проекты:

  • NuPIC (Numenta Platform for Intelligent Computing, https://numenta.org). Имплементация HTM, фреймворк, написан на C++ и Python. Лицензия AGPLv3, есть и триал

Наиболее значимыми можно считать Grok и Cortical.io. Numenta имеет также несколько десятков патентов. Имеет разработанную лицензионную политику

Biological and Machine Intelligence (BAMI) - "живая" книга ( https://numenta.com/resources/biological-and-machine-intelligence)
